Variáveis em Python

Observação
  • Variáveis não são caixas em Python, mas sim labels
  • A qualquer momento um label pode ser associado a um objeto diferente
  • Múltiplos labels podem estar associados ao mesmo objeto
a = [1, 2, 3] # Cria um objeto (uma lista) e associa ao label 'a' b = a # Associa a lista com um outro label, 'b' -> não há cópia! a is b # a e b não são apenas duas listas iguais, eles são o mesmo objeto True a[0] = 100 # Modifica o objeto chamado de 'a' print(b) # 'b' foi modificado pois é o mesmo objeto de 'a' [100, 2, 3] a = [100, 2, 3] # Cria um novo objeto e associa ao label 'a' a == b # Atualmente os objetos referenciados por 'a' e 'b' True a is b # ... mas eles não são o mesmo objeto -> Podemos mudar um sem mudar o outro False
