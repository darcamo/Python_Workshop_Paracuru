Unit Tests

Definição
Tipo particular de testes correspondendo a menor parte testável de um programa
Características
  • Idealmente cada unidade de teste deve ser independente das demais
  • Cada teste deve poder rodar de maneira isolada ou como parte de uma suíte de testes
    • A ordem de execução dos testes não deve importar
    • Cada teste deve importar seu próprio conjunto de dados independentes dos demais testes
Importante
  • Os testes devem rodar rapidamente e de maneira automatizada
  • Ao identificar um bug no seu programa, crie um teste que reproduza o bug só então conserte o bug quando o teste passar o bug foi corrigido
